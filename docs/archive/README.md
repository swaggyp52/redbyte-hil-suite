reflect the current product positioning.

- Root **`README.md`** — product overview, workflow, modules, honesty statement.
- **`docs/PROJECT_OVERVIEW.md`** — capstone framing.
- **`docs/architecture.md`** and **`docs/api_reference.md`** — engineering reference.
- **`docs/demo_script.md`** and **`docs/presentation_walkthrough.md`** — demo flow.
- **`docs/test_plan.md`**, **`docs/fmea.md`**, **`docs/HARDWARE_INTEGRATION.md`**,
  **`docs/protocol.md`** — engineering artifacts used by the final paper.
