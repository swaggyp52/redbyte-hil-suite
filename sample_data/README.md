# Sample Data

These files are bundled with the GitHub ZIP so a fresh Windows install can prove the package works without any private project data.

- `demo_three_phase.csv` - small Rigol-style three-phase waveform sample
- `demo_session.json` - small saved session capsule that can be imported directly

Use **Import Run File** in the app and choose either file.

The larger senior-design validation files are optional and are not required for install or smoke validation.
